# 🔄 Workflow: Синхронизация БД (Supabase Sync)

Для того чтобы TypeScript понимал структуру базы данных, нужно периодически обновлять типы.

## 1. Обновление схемы
- [ ] Примени изменения в Supabase Dashboard или через миграции.
- [ ] Убедись, что RLS политики настроены верно для анонимных и авторизованных пользователей.

## 2. Генерация типов
Запусти команду для генерации типов из живой базы данных:
- [ ] Используй инструмент `mcp_supabase-mcp-server_generate_typescript_types`.
- [ ] Или выполни команду в терминале:
  ```bash
  npx supabase gen types typescript --project-id YOUR_PROJECT_ID > src/types/supabase.ts
  ```

## 3. Проверка интеграции
- [ ] Проверь `src/lib/supabase.ts` на соответствие новым типам.
- [ ] Убедись, что запросы в `useNodesStore` не подсвечиваются ошибками TS.

## 4. Обновление Django (если нужно)
- [ ] Если структура изменилась значительно, обнови модели в Django.
- [ ] Проверь SSL-соединение и Service Role Key.

---
> [!IMPORTANT]
> Никогда не используй `Service Role Key` на фронтенде. Только в Django.
